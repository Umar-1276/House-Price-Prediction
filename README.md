# House Price Prediction

## 📌 Project Overview
This project predicts house prices using **XGBoost Regression**, a powerful gradient boosting algorithm for regression problems.

## 📂 Dataset
- Housing Price Dataset
- Target: House Price
- Features: Various house attributes (such as area, bedrooms, bathrooms, etc., depending on the dataset)

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- XGBoost
- Jupyter Notebook / VS Code

## 🤖 Machine Learning Model
- XGBoost Regressor

## 📊 Workflow
1. Import libraries
2. Load the dataset
3. Data preprocessing
4. Split features and target
5. Train-test split
6. Train the XGBoost Regression model
7. Evaluate the model
8. Predict house prices

## 📈 Model Evaluation
Evaluation metrics used:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score (if used)

## 🚀 How to Run

1. Clone this repository

2. Install the required libraries

```bash
pip install numpy pandas scikit-learn xgboost
```

3. Open the notebook and run all cells.
